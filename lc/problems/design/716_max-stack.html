Reference: LeetCode
Difficulty: Easy

Problem

Design a max stack that supports push, pop, top, peekMax and popMax.

Note:

  • -1e7 <= x <= 1e7
  • Number of operations won’t exceed 10000.
  • The last four operations won’t be called when stack is empty.
